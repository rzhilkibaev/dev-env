dev-env
=======

!macOS only!

## Usage
`./install PACKAGE...`

## Tools
Alacritty
Zsh
tmux
nvim
fzf
ag
pack/unpack

## Directory structure
```
root
├── install # main script
└── packages # directory with installable packages
    └── <package_name>
        └── install # package installation script
```
